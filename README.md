# Disk Manager

A comprehensive disk management application featuring a Flutter frontend and a Rust backend.

## Features
- File and folder management.
- Media preview (images, videos, text).
- Platform: Windows (and others supported by Flutter).

## Project Structure
- `lib/`: Flutter frontend source code.
- `backed/`: Rust backend source code.

## Getting Started

### Prerequisites
- Flutter SDK
- Rust (Cargo)

### Running

1. **Start the Backend**
   ```bash
   cd backed
   cargo run
   ```

2. **Start the Frontend**
   ```bash
   # Open a new terminal
   flutter run -d windows
   ```
